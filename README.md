# 🚀 Telegram-Connected Web App

A clean, modern web interface that seamlessly integrates with a Telegram Bot. This project demonstrates how to capture user interaction on a website and securely route that data directly to a Telegram chat using the Telegram Bot API.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (with dynamic Telegram Theme variable support)
* **Logic & Integration:** JavaScript (Vanilla JS / Fetch API)
* **API:** Telegram Bot API

---

## ✨ Features

* **Instant Messaging:** Sends website form data or user interactions straight to your Telegram app in real-time.
* **Telegram Mini App Ready:** Includes CSS styling variables (`var(--tg-theme-...)`) that automatically adapt to Telegram's light or dark mode if opened inside the app.
* **Responsive Design:** A beautifully styled layout built entirely from scratch with semantic HTML and modern CSS.

---

## 📦 Installation & Setup

Follow these steps to get a local copy of this project up and running.

### 1. Prerequisites
You will need a Telegram Bot Token. 
* Open Telegram and search for [@BotFather](https://t.me/BotFather).
* Use the `/newbot` command to create a bot and copy your **API Token**.
* Message [@userinfobot](https://t.me/userinfobot) to find your personal **Chat ID**.
